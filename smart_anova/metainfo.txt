Name of Quantlet: SPL_smart_anova

Published in: Statistical Programming Languages

Description: Variance Analysis between groups, uses Anova if the distributions within groups are normal, in the opposite case, uses Kruskall Wallis test

Keywords: test, anova

Author:

Submitted:

Datafile:

Input: numeric variable to test, a categorical variable for the group partitioning, and the alpha significancy level, which is by default 0.05

Output: p-value of the test and the textual message containing the answer on whether the groups are significantly different from each other ot not for a given alpha

Example: data("ToothGrowth"); smart_anova(ToothGrowth$len, ToothGrowth$supp)
